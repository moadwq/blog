# 个人博客
一个简单的 vue + springboot 前后端分离个人博客项目

# 使用技术
jdk：8

mysql：5.7

redis：6.2

springboot：2.6.12

mybatisPlus：3.4.3

springfox-swagger2：3.0.0

easyexcel：3.0.5

七牛云oss：[7.7.0, 7.10.99]
